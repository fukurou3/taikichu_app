**緊急停止スクリプト**
&nbsp; ✅ 機能:
&nbsp; - Cloud Run サービストラフィック停止
&nbsp; - Redis インスタンス削除（コスト削減最重要）
&nbsp; - Firebase Functions 無効化
&nbsp; - 安全確認プロンプト付き

  **使用方法:**
&nbsp; ./緊急停止スクリプト.sh

  **手動緊急停止コマンド**
&nbsp; # Analytics Service停止
&nbsp; gcloud run services update analytics-service --region=asia-northeast1 --no-traffic   

&nbsp; # Redis削除
&nbsp; gcloud redis instances delete taikichu-redis-test --region=asia-northeast1 --quiet  

&nbsp; # 全Docker停止
&nbsp; docker-compose down



  **テスト実行**
&nbsp; Flutter テスト
&nbsp; # ローカル環境用
&nbsp; flutter run --dart-define=ENVIRONMENT=development

&nbsp; # テスト実行
&nbsp; flutter test
&nbsp; flutter test integration\_test/

&nbsp; # 環境の健全性確認
&nbsp; ./scripts/health-check.sh

**クラウド環境の「正しい」停止方法**
Memorystore for Redis存在しているだけで時間単位の料金が発生し続ける。
\# Redisインスタンスを削除して課金を停止する
gcloud redis instances delete taikichu-redis --region=asia-northeast1

\# 次のテストの前にRedisインスタンスを再作成する
gcloud redis instances create taikichu-redis --size=1 --region=asia-northeast1 --redis-version=redis\_6\_x

\# 外部からの未認証アクセスをすべて拒否して、サービスを事実上停止させる
gcloud run services update analytics-service --no-allow-unauthenticated --region=asia-northeast1
